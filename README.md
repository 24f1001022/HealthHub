# HealthHub

Hospital Management System — patients, doctors, appointments, treatments, CSV export, and email notifications.

## Live deployment

Follow **[DEPLOY.md](./DEPLOY.md)** for free hosting on Vercel + Render + Upstash.

## Tech stack

- **Frontend:** Vue 3, Vite, Vue Router, Vuex, Axios
- **Backend:** Flask, SQLAlchemy, Redis sessions/cache, Celery
- **Email:** Flask-Mail (Gmail SMTP)

## Default admin

After first deploy: `admin@admin.com` / `admin` — change in production if needed.
